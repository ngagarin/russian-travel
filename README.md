# Проект: Путешествие по России

## Описание проекта:

"Rissian-travel" - Адаптивный, одностраничный сайт о путешествии по России.

При создании сайта использовались следующие инструменты:
1. Flex-box
2. Grid
3. Трансформации
4. Методология БЭМ
5. Использование GIT
